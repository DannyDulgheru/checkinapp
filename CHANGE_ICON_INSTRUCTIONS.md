# Instructions to Change App Icon

## Icon Files Needed

Place your new icon image in these locations with these sizes:

1. **`public/icons/icon-192.png`** - 192x192 pixels
2. **`public/icons/icon-512.png`** - 512x512 pixels  
3. **`public/icons/favicon.png`** - 32x32 or 16x16 pixels
4. **`public/icons/apple-touch-icon.png`** - 180x180 pixels
5. **`public/icons/icon.png`** - Any size (backup)

## Steps

1. Save your icon image
2. Resize it to the required sizes (192, 512, 180, and 32/16)
3. Replace the files in `public/icons/` folder
4. Restart the dev server

## If you have one large image

If you only have one large image (e.g., 512x512), I can help you create all the sizes needed.
